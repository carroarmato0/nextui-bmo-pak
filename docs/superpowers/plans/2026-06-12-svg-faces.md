# SVG Face Rendering & Override-or-Default Assets — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the procedural face rasterizer with anti-aliased SVG-rendered faces (embedded defaults, optional on-disk `faces/` mod overrides), and extend the same override-or-default model to persona/voice/quotes.

**Architecture:** New pure-Go package `internal/face` parses and rasterizes full-scene 280×210 SVGs with `oksvg`+`rasterx` into ARGB pixel buffers, cached per resolution. The SDL renderer blits the cached face and draws dynamic extras (ZZZ, clock, overlay) on top. Speaking is a Go-templated SVG rendered at 12 quantized mouth-openness levels (base frame + mouth-band strips). Prompt files become read-if-present overrides; the app never creates them.

**Tech Stack:** Go 1.25, `github.com/srwiley/oksvg` + `github.com/srwiley/rasterx` (pure Go), `go:embed`, `text/template`, go-sdl2 (unchanged).

**Spec:** `docs/superpowers/specs/2026-06-12-svg-faces-design.md`

## Canonical SVG geometry (used throughout — derived from the bmo-face skill)

ViewBox `0 0 280 210`. Body teal `#4ECBA8`, screen `#90e5c8` at x=22 y=20 w=202 h=155 rx=14, dark `#1a1a1a`, teeth `#e4e4e4`, mouth interior `#1a7848`, tongue `#16ae81`. Eyes: left cx=63, right cx=182, cy=78.

The bmo-face skill's open/speaking mouths use `clipPath`, which **oksvg does not support**. They are rewritten here as explicit paths — the teeth band and interior are the exact geometric intersection of the original bands with the rx-rounded mouth rect (corner-arc intersections computed with `dx = √(r² − (r−bandH)²)`); the tongue is the upper half-ellipse, which stays inside the mouth at canonical sizes.

Every face file starts with this boilerplate:

```svg

  <rect x="0" y="0" width="280" height="210" fill="#4ECBA8"/>
  <rect x="22" y="20" width="202" height="155" rx="14" fill="#90e5c8"/>
  <!-- face elements -->
</svg>
```

```go
// Package face loads BMO's facial expressions from SVG assets and rasterizes
// them into ARGB8888 pixel buffers for the SDL renderer. Defaults are embedded
// in the binary; an optional faces/ directory on disk overrides them per file.
package face

import "strings"

// Canonical expression names — one embedded SVG asset exists per name.
const (
	ExprNeutral   = "neutral"
	ExprBlink     = "blink"
	ExprListening = "listening"
	ExprThinking  = "thinking"
	ExprSpeaking  = "speaking"
	ExprSleeping  = "sleeping"
	ExprConcerned = "concerned"
	ExprSmile     = "smile"
)

// CanonicalNames lists every canonical expression, used for warm-up and tests.
var CanonicalNames = []string{
	ExprNeutral, ExprBlink, ExprListening, ExprThinking,
	ExprSpeaking, ExprSleeping, ExprConcerned, ExprSmile,
}

// Canonical maps any expression alias the assistant emits to the canonical
// asset name. Unknown names render as neutral.
func Canonical(expr string) string {
	switch strings.ToLower(strings.TrimSpace(expr)) {
	case "", "idle", ExprNeutral:
		return ExprNeutral
	case ExprBlink:
		return ExprBlink
	case "asleep", "sleep", ExprSleeping:
		return ExprSleeping
	case "error", "confused", "angry", "sad", ExprConcerned:
		return ExprConcerned
	case "happy", "laugh", "excited", ExprSmile:
		return ExprSmile
	case ExprListening:
		return ExprListening
	case ExprThinking:
		return ExprThinking
	case ExprSpeaking:
		return ExprSpeaking
	default:
		return ExprNeutral
	}
}
```

```go
// argbAt returns the (r,g,b) of the pixel nearest the given viewBox coordinate
// in a w×h buffer rendered from the 280×210 viewBox (non-uniform stretch).
func argbAt(buf []uint32, w, h int, vx, vy float64) (uint8, uint8, uint8) {
	x := int(vx / 280.0 * float64(w))
	y := int(vy / 210.0 * float64(h))
	px := buf[y*w+x]
	return uint8(px >> 16), uint8(px >> 8), uint8(px)
}

func near(a, b uint8) bool {
	d := int(a) - int(b)
	return d >= -8 && d <= 8
}

func assertColor(t *testing.T, buf []uint32, w, h int, vx, vy float64, wr, wg, wb uint8, what string) {
	t.Helper()
	r, g, b := argbAt(buf, w, h, vx, vy)
	if !near(r, wr) || !near(g, wg) || !near(b, wb) {
		t.Errorf("%s at viewBox(%.1f,%.1f): got #%02x%02x%02x want #%02x%02x%02x", what, vx, vy, r, g, b, wr, wg, wb)
	}
}

func TestRasterizeStretchesToFill(t *testing.T) {
	// 1280x720 is 16:9 against the 4:3 viewBox: the circle must land at the
	// stretched position, proving non-uniform scaling (no letterboxing).
	for _, size := range [][2]int{{1024, 768}, {1280, 720}} {

		if err != nil {
			t.Fatalf("Rasterize(%dx%d): %v", w, h, err)
		}

			t.Fatalf("buffer length %d, want %d", len(buf), w*h)

		assertColor(t, buf, w, h, 5, 5, 0x4e, 0xcb, 0xa8, "background corner")
		assertColor(t, buf, w, h, 275, 205, 0x4e, 0xcb, 0xa8, "background corner")
```

```go
package face

import (
	"bytes"
	"fmt"
	"image"

	"github.com/srwiley/oksvg"
	"github.com/srwiley/rasterx"
)

// Rasterize renders SVG bytes into a w×h ARGB8888 pixel buffer (row-major,
// stride == w), stretching the viewBox non-uniformly to fill the full target —
// BMO's body always covers the whole screen regardless of aspect ratio.
func Rasterize(svg []byte, w, h int) ([]uint32, error) {
	if w <= 0 || h <= 0 {
		return nil, fmt.Errorf("invalid raster size %dx%d", w, h)
	}
	icon, err := oksvg.ReadIconStream(bytes.NewReader(svg), oksvg.WarnErrorMode)
	if err != nil {
		return nil, fmt.Errorf("parse svg: %w", err)
	}
	icon.SetTarget(0, 0, float64(w), float64(h))
	img := image.NewRGBA(image.Rect(0, 0, w, h))
	scanner := rasterx.NewScannerGV(w, h, img, img.Bounds())
	icon.Draw(rasterx.NewDasher(w, h, scanner), 1.0)

	out := make([]uint32, w*h)
	pix := img.Pix
	for i := range out {
		o := i * 4
		out[i] = uint32(pix[o+3])<<24 | uint32(pix[o])<<16 | uint32(pix[o+1])<<8 | uint32(pix[o+2])
	}
	return out, nil
}
```

**Contingency:** if `TestRasterizeStretchesToFill` fails because oksvg's `SetTarget` preserves aspect ratio (letterboxes), render instead at the viewBox's own aspect and verify against oksvg's `SvgIcon.Transform` docs — but per oksvg source, `SetTarget` scales X and Y independently, so this is expected to pass as written. Do not silently weaken the test; the stretch behavior is a spec requirement.

Note: `oksvg.ReadIconStream` may not error on plain garbage (it tolerates broken XML in `WarnErrorMode`). If `TestRasterizeRejectsGarbage` fails on the first case, switch to `oksvg.StrictErrorMode` for parsing. Strict mode is correct here: a mod file that doesn't parse must trigger the fallback path, not render garbage.

```go
import (
	"bytes"
	"os"
	"path/filepath"
	"regexp"
	"strings"

// Library resolves expression names to SVG bytes. Embedded defaults are the
// source of truth; files in an optional override directory win per file. The
// app never creates or writes the directory.
type Library struct {
	dir  string
	logf func(string, ...any)
}

// NewLibrary returns a Library reading overrides from dir (may not exist).
func NewLibrary(dir string) *Library {
	return &Library{dir: dir, logf: func(string, ...any) {}}
}

// SetLogf installs a logger for override-load warnings (e.g. logger.Warnf).
func (l *Library) SetLogf(f func(string, ...any)) {
	if f != nil {
		l.logf = f
	}
}

// fileNameRe restricts override lookups to safe simple names.
var fileNameRe = regexp.MustCompile(`^[a-z0-9_-]+$`)

// Bytes returns the SVG bytes for expr and whether they came from disk.
// Lookup order: faces/<name>.svg → faces/<canonical>.svg → embedded canonical
// → embedded neutral (always present).
func (l *Library) Bytes(expr string) ([]byte, bool) {
	raw := strings.ToLower(strings.TrimSpace(expr))
	canonical := Canonical(raw)
	if l.dir != "" {
		names := []string{raw}
		if canonical != raw {
			names = append(names, canonical)
		}
		for _, name := range names {
			if !fileNameRe.MatchString(name) {
				continue
			}
			data, err := os.ReadFile(filepath.Join(l.dir, name+".svg"))
			if err != nil {
				continue
			}
			if len(bytes.TrimSpace(data)) == 0 {
				l.logf("face override %s.svg is blank; using default", name)
				continue
			}
			return data, true
		}
	}
	if data, ok := defaultBytes(canonical); ok {
		return data, false
	}
	data, _ := defaultBytes(ExprNeutral)
	return data, false
}
```

```go
package face

import (
	"bytes"
	"fmt"
	"math"
	"text/template"
)

// speakLevels is the number of quantized mouth-openness frames.
const speakLevels = 12

// SpeakParams is the data passed to the speaking.svg template. Documented in
// docs/FACES.md — mod templates may use any subset of these fields.
type SpeakParams struct {
	MouthH  float64 // mouth rect height in viewBox units (6..36)
	MouthRx float64 // mouth rect corner radius
	TeethPath    string
	InteriorPath string
	TonguePath   string
}

// IsSpeakTemplate reports whether a speaking.svg file is a Go template
// (animated) rather than a plain static SVG.
func IsSpeakTemplate(data []byte) bool {
	return bytes.Contains(data, []byte("{{"))
}

// speakParams computes all template values for openness t in [0,1].
func speakParams(t float64) SpeakParams {
	t = math.Max(0, math.Min(1, t))
	const left, right, top = 89.0, 157.0, 106.0
	h := 6 + 30*t
	r := math.Min(16, h/2)
	bottom := top + h

	// Teeth band: height 0.28h < r always, so its bottom edge crosses the
	// top-corner arcs at horizontal offset dx from the rect edges.
	th := 0.28 * h
	dy := r - th
	dx := math.Sqrt(r*r - dy*dy)
	tlx, trx := left+r-dx, right-r+dx
	tby := top + th

	teeth := fmt.Sprintf(
		"M %.2f %.2f A %.2f %.2f 0 0 1 %.2f %.2f L %.2f %.2f A %.2f %.2f 0 0 1 %.2f %.2f Z",
		tlx, tby, r, r, left+r, top, right-r, top, r, r, trx, tby)

	interior := fmt.Sprintf(
		"M %.2f %.2f L %.2f %.2f "+
			"A %.2f %.2f 0 0 1 %.2f %.2f L %.2f %.2f "+
			"A %.2f %.2f 0 0 1 %.2f %.2f L %.2f %.2f "+
			"A %.2f %.2f 0 0 1 %.2f %.2f L %.2f %.2f "+
			"A %.2f %.2f 0 0 1 %.2f %.2f Z",
		tlx, tby, trx, tby,
		r, r, right, top+r, right, bottom-r,
		r, r, right-r, bottom, left+r, bottom,
		r, r, left, bottom-r, left, top+r,
		r, r, tlx, tby)

	// Tongue: upper half-ellipse centered on the mouth's bottom edge,
	// scaled with the opening (skill rule: rx≈0.28·w, ry≈0.18·h).
	tr := 19.0 * h / 36.0
	ty := 0.18 * h
	tongue := fmt.Sprintf("M %.2f %.2f A %.2f %.2f 0 0 1 %.2f %.2f Z",
		123-tr, bottom, tr, ty, 123+tr, bottom)

	return SpeakParams{
		MouthH:       h,
		MouthRx:      r,
		TeethPath:    teeth,
		InteriorPath: interior,
		TonguePath:   tongue,
	}
}

// renderSpeakSVG executes a speaking template at openness t and returns the
// resulting SVG document.
func renderSpeakSVG(tmplData []byte, t float64) ([]byte, error) {
	tmpl, err := template.New("speaking").Parse(string(tmplData))
	if err != nil {
		return nil, fmt.Errorf("parse speaking template: %w", err)
	}
	var buf bytes.Buffer
	if err := tmpl.Execute(&buf, speakParams(t)); err != nil {
		return nil, fmt.Errorf("execute speaking template: %w", err)
	}
	return buf.Bytes(), nil
}
```

```go
package face

import "sync"

// Strip is a rectangular sub-region of a frame (the speaking mouth band).
type Strip struct {
	X, Y, W, H int
	Pix        []uint32 // row-major, stride == W
}

// speakSet holds the rendered speaking faces for one resolution: either a
// single static frame (mod-provided plain SVG) or a base frame plus one
// mouth-band strip per openness level above zero.
type speakSet struct {
	static []uint32
	base   []uint32
	strips []*Strip
}

// Cache rasterizes and caches face frames per resolution. Safe for concurrent
// use; a background Warm call may rasterize while the render loop reads.
type Cache struct {
	lib    *Library
	mu     sync.Mutex
	w, h   int
	frames map[string][]uint32
	failed map[string]bool
	speak  *speakSet
	speakFailed bool
}

// NewCache returns a Cache backed by lib.
func NewCache(lib *Library) *Cache {
	return &Cache{lib: lib}
}

// Warm pre-rasterizes every canonical expression at w×h. Call from a
// goroutine at startup; later Frame/Speak calls then hit the cache.
func (c *Cache) Warm(w, h int) {
	for _, name := range CanonicalNames {
		if name == ExprSpeaking {
			continue
		}
		c.Frame(name, w, h)
	}
	c.Speak(1, w, h)
}

// Frame returns the cached ARGB buffer for expr at w×h, rasterizing on first
// use. Returns nil only if both the override and the embedded default fail.
func (c *Cache) Frame(expr string, w, h int) []uint32 {
	canonical := Canonical(expr)
	c.mu.Lock()
	defer c.mu.Unlock()
	c.resizeLocked(w, h)
	if buf, ok := c.frames[canonical]; ok {
		return buf
	}
	if c.failed[canonical] {
		return nil
	}
	buf := c.renderLocked(canonical, w, h)
	if buf == nil {
		c.failed[canonical] = true
		return nil
	}
	c.frames[canonical] = buf
	return buf
}

// Speak returns the speaking base frame and, for openness t in (0,1] with an
// animated template, the mouth strip to blit over it. Static speaking faces
// return (frame, nil).
func (c *Cache) Speak(t float64, w, h int) ([]uint32, *Strip) {
	c.mu.Lock()
	defer c.mu.Unlock()
	c.resizeLocked(w, h)
	if c.speak == nil && !c.speakFailed {
		c.speak = c.buildSpeakLocked(w, h)
		c.speakFailed = c.speak == nil
	}
	s := c.speak
	if s == nil {
		return nil, nil
	}
	if s.static != nil {
		return s.static, nil
	}
	if t < 0 {
		t = 0
	}
	if t > 1 {
		t = 1
	}
	lvl := int(t*float64(speakLevels-1) + 0.5)
	if lvl <= 0 {
		return s.base, nil
	}
	return s.base, s.strips[lvl-1]
}

// resizeLocked drops all cached frames when the output size changes.
func (c *Cache) resizeLocked(w, h int) {
	if w == c.w && h == c.h && c.frames != nil {
		return
	}
	c.w, c.h = w, h
	c.frames = make(map[string][]uint32)
	c.failed = make(map[string]bool)
	c.speak = nil
	c.speakFailed = false
}

// renderLocked rasterizes canonical at w×h, falling back from a broken disk
// override to the embedded default.
func (c *Cache) renderLocked(canonical string, w, h int) []uint32 {
	data, fromDisk := c.lib.Bytes(canonical)
	buf, err := Rasterize(data, w, h)
	if err == nil {
		return buf
	}
	c.lib.logf("face %s: %v", canonical, err)
	if fromDisk {
		if def, ok := defaultBytes(canonical); ok {
			if buf, derr := Rasterize(def, w, h); derr == nil {
				return buf
			}
		}
	}
	return nil
}

// buildSpeakLocked renders the speaking face: animated levels for a template,
// a static frame for a plain SVG, with template→embedded fallback.
func (c *Cache) buildSpeakLocked(w, h int) *speakSet {
	data, fromDisk := c.lib.Bytes(ExprSpeaking)
	if !IsSpeakTemplate(data) {
		if buf, err := Rasterize(data, w, h); err == nil {
			return &speakSet{static: buf}
		}
		c.lib.logf("face speaking: static override failed; using default template")
		if fromDisk {
			data, _ = defaultBytes(ExprSpeaking)
		} else {
			return nil
		}
	}
	set := c.renderSpeakLevelsLocked(data, w, h)
	if set == nil && fromDisk {
		c.lib.logf("face speaking: override template failed; using default template")
		if def, ok := defaultBytes(ExprSpeaking); ok {
			set = c.renderSpeakLevelsLocked(def, w, h)
		}
	}
	return set
}

// renderSpeakLevelsLocked rasterizes all openness levels from a template and
// extracts mouth-band strips for levels above zero.
func (c *Cache) renderSpeakLevelsLocked(tmplData []byte, w, h int) *speakSet {
	bx0, by0, bx1, by1 := speakBand(w, h)
	set := &speakSet{}
	for lvl := 0; lvl < speakLevels; lvl++ {
		t := float64(lvl) / float64(speakLevels-1)
		svg, err := renderSpeakSVG(tmplData, t)
		if err != nil {
			c.lib.logf("face speaking level %d: %v", lvl, err)
			return nil
		}
		buf, err := Rasterize(svg, w, h)
		if err != nil {
			c.lib.logf("face speaking level %d: %v", lvl, err)
			return nil
		}
		if lvl == 0 {
			set.base = buf
			continue
		}
		sw, sh := bx1-bx0, by1-by0
		strip := &Strip{X: bx0, Y: by0, W: sw, H: sh, Pix: make([]uint32, sw*sh)}
		for row := 0; row < sh; row++ {
			copy(strip.Pix[row*sw:(row+1)*sw], buf[(by0+row)*w+bx0:(by0+row)*w+bx1])
		}
		set.strips = append(set.strips, strip)
	}
	return set
}

// speakBand returns the pixel bounds of the mouth region (viewBox x 75..171,
// y 96..150 — the speaking mouth plus an anti-aliasing margin).
func speakBand(w, h int) (x0, y0, x1, y1 int) {
	x0 = int(75.0 / 280.0 * float64(w))
	y0 = int(96.0 / 210.0 * float64(h))
	x1 = int(171.0/280.0*float64(w)) + 1
	y1 = int(150.0/210.0*float64(h)) + 1
	if x1 > w {
		x1 = w
	}
	if y1 > h {
		y1 = h
	}
	return x0, y0, x1, y1
}
```

```go
// blitFace copies the SVG-rendered face for the current frame into the pixel
// buffer; reports false when no frame is available.
func (r *Renderer) blitFace(canonical string, frame FrameState, phase float64) bool {
	if r.faces == nil {
		return false
	}
	w, h := int(r.W), int(r.H)
	if canonical == face.ExprSpeaking {
		// Amplitude drives mouth openness; without amplitude data fall back
		// to the legacy phase oscillation so TTS still animates.
		t := math.Sqrt(math.Min(1, float64(frame.SpeakAmplitude)))
		if frame.SpeakAmplitude <= 0 {
			t = 0.45 + 0.35*math.Sin(phase*8.0)
		}
		base, strip := r.faces.Speak(t, w, h)
		if base == nil || len(base) != len(r.pixels) {
			return false
		}
		copy(r.pixels, base)
		if strip != nil {
			r.blitStrip(strip)
		}
		return true
	}
	buf := r.faces.Frame(canonical, w, h)
	if buf == nil || len(buf) != len(r.pixels) {
		return false
	}
	copy(r.pixels, buf)
	return true
}

// blitStrip copies a face sub-region (the speaking mouth band) into place.
func (r *Renderer) blitStrip(s *face.Strip) {
	if s == nil || int32(s.X+s.W) > r.W || int32(s.Y+s.H) > r.H || len(s.Pix) < s.W*s.H {
		return
	}
	for row := 0; row < s.H; row++ {
		off := (s.Y+row)*r.stride + s.X
		copy(r.pixels[off:off+s.W], s.Pix[row*s.W:(row+1)*s.W])
	}
}

	r.fillRoundedRect(inner.X, inner.Y, inner.W, inner.H, layout.CornerRadius-layout.ScreenInset/2, rgba{0x90, 0xe5, 0xc8, 255})
```

```go
// LoadPromptFile returns the override file's trimmed content when it exists
// and is non-blank, and def otherwise. The file is never created or written:
// built-in defaults ship with the binary and apply wherever no override
// exists, so updated defaults reach every install that hasn't overridden them.
func LoadPromptFile(path, def string) string {
	if data, err := os.ReadFile(path); err == nil {
		if content := strings.TrimSpace(string(data)); content != "" {
			return content
		}
	}
	return strings.TrimSpace(def)
}

// RemoveOverrides deletes override files so the built-in defaults apply
// again; wired to the settings menu RESTORE DEFAULTS item. Missing files are
// not an error.
func RemoveOverrides(paths ...string) error {
	var firstErr error
	for _, p := range paths {
		if err := os.Remove(p); err != nil && !errors.Is(err, os.ErrNotExist) && firstErr == nil {
			firstErr = fmt.Errorf("remove override %s: %w", p, err)
		}
	}
	return firstErr
}
```

```markdown
# Modding BMO: Faces, Persona, Voice, Quotes

BMO's look and personality are fully moddable without recompiling. All
override files live in the BMO data directory (the same folder that holds
BMO's config and `memory.json`):

    <dataRoot>/BMO/
      persona.txt      # chat persona sent to the LLM
      voice.txt        # TTS speaking-style instructions
      quotes.txt       # idle quotes, one per line
      faces/           # expression SVGs (see below)

**Built-in defaults always ship with the app.** A file here *overrides* the
default; delete the file (or use Settings → RESTORE DEFAULTS for the text
files) to go back to the built-ins and automatically receive future updates.
The app never creates or modifies these files. A mod pack is just a zip of
this layout.

## Faces

One SVG per expression, full scene, `viewBox="0 0 280 210"`:

| File | Shown when |
|------|------------|
| `neutral.svg` | idle default |
| `blink.svg` | idle blink |
| `listening.svg` | push-to-talk recording |
| `thinking.svg` | waiting for the LLM |
| `speaking.svg` | TTS playback (see below) |
| `sleeping.svg` | quota exhausted (ZZZ marks are drawn on top) |
| `concerned.svg` | errors / setup required |
| `smile.svg` | happy / laugh / excited |

Any subset may be provided; missing files use the built-in default. Alias
filenames also work (`laugh.svg`, `excited.svg`, `happy.svg`, `error.svg`,
…) and win over the canonical file for that alias.

The scene is stretched to fill the screen (non-uniformly on 16:9 devices).
The corner clock, sleep ZZZ marks, and settings menu are drawn on top by the
app.

### Supported SVG subset

The renderer is a partial SVG implementation (oksvg). Supported: `path`
(all commands), `rect`, `circle`, `ellipse`, `line`, `polygon`, `polyline`,
`g`, `defs`/`use`, `transform`, fill/stroke/opacity, linear and radial
gradients. **Not supported:** `clipPath`, masks, filters, `text`, CSS
classes/stylesheets, `pattern`, embedded images. A file that fails to parse
is logged and the built-in default is used instead.

### speaking.svg: animated vs static

The built-in `speaking.svg` is a Go text template: the app renders it at 12
mouth-openness levels and animates them with the audio amplitude. Template
parameters (all floats/path strings, openness 0..1):

- `{{.MouthH}}` — mouth rect height (6–36 viewBox units)
- `{{.MouthRx}}` — mouth rect corner radius
- `{{.TeethPath}}`, `{{.InteriorPath}}`, `{{.TonguePath}}` — precomputed
  path data for the default mouth construction

A modded `speaking.svg` **without** `{{` markers is treated as a plain
static face during speech — that is the expected, supported mod path. A
broken template falls back to the built-in one.

## Persona, voice, quotes

Plain UTF-8 text. `persona.txt` is the system prompt; `voice.txt` is sent as
TTS style instructions on instruction-capable models; `quotes.txt` is one
idle quote per line. Blank files are treated as absent. Changes to persona
and voice are picked up before each utterance — no restart needed.
```
